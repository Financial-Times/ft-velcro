Browser support

Specific details about the level of support for different browsers are available elsewhere.

Graceful degradation

To ensure maximum performance and low maintenance, native CSS is used as much as possible. This means that in older browsers there may be minor differences in appearance. These differences should be listed in this document alongside the examples.

General examples of degradation:

  • The Benton font may be replaced with Helvetica or Arial

Interaction states

Standard class names will be used to mark the interaction states of page elements. These classes are not styled on their own, but instead used in combination with other classes.

  • :hover - when mouse pointer is positioned over the element
  • :focus - when the element has focus
  • ft-state-selected e.g. for a tab when its contents are currently shown
  • ft-state-disabled for when something is currently not functional
  • ft-state-loading

Spacing

Velcro text styles do not include any margins to create space between them and other elements.

Spacing is applied using the following classes:

  • ft-spc-btm-full one full unit of spacing below
  • ft-spc-btm-half half unit of spacing below
  • ft-spc-btm-qtr quarter unit of spacing below
